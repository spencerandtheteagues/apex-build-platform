Build a production-ready team messaging app called "Threadline".

It is a Slack-style workspace for project teams with channels, direct messages, threads, and unread state. Use React + TypeScript + Tailwind CSS + shadcn/ui. Store all data in memory and simulate realtime locally without external APIs.

Must include:
- Workspace shell with channel sidebar, direct messages, unread badges, search, and user presence.
- Channel view with message list, composer, attachments simulation, reactions, pinned messages, and date separators.
- Thread panel with replies, mention suggestions, resolved/unresolved status, and follow toggle.
- Direct message view with typing indicator simulation, send/edit/delete, and unread tracking.
- Search page with filters for channel, author, date, and result highlighting.
- Settings page with notification preferences, workspace members, roles, and reset demo data.

Use realistic conversation data, smooth scrolling behavior, keyboard-friendly composer interactions, responsive layout, and no console errors.
