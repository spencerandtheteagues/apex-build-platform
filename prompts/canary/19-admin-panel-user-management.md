Build a production-ready admin panel called "AccessDesk".

It is an internal user and permissions management console for a B2B SaaS company. Use React + TypeScript + Tailwind CSS + shadcn/ui. Store all data in memory and do not use external APIs.

Must include:
- Admin dashboard with user counts, active sessions, risky accounts, pending invites, and audit summary.
- Users table with search, role/status filters, pagination simulation, bulk actions, and user detail drawer.
- User detail view with profile, teams, permissions, recent activity, sessions, and suspend/reactivate actions.
- Roles and permissions page with permission matrix, role editor, unsaved-change warning, and preview mode.
- Audit log with event filters, severity chips, actor/resource fields, and export simulation.
- Settings page with SSO placeholders, retention policy, security toggles, and reset demo data.

Use dense but readable operational UI, confirmations for destructive actions, validation, responsive layout, loading/empty states, and no console errors.
