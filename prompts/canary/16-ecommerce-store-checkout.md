Build a production-ready ecommerce storefront called "Northline Goods".

It is a curated outdoor gear shop with product discovery, cart, and simulated checkout. Use React + TypeScript + Tailwind CSS + shadcn/ui. Store all data in memory, simulate payment locally, and do not use real payments or external APIs.

Must include:
- Storefront home with featured collections, promotional banner, trust badges, and product highlights.
- Product catalog with search, category filters, price filter, sort, ratings, stock status, and quick add.
- Product detail view with gallery, variants, quantity selector, reviews, shipping estimate, and related products.
- Cart drawer/page with quantity updates, promo code simulation, tax/shipping calculation, and order summary.
- Checkout flow with contact, shipping, delivery, payment placeholder, validation, and order confirmation.
- Account/orders page with order history, return request simulation, and reset demo data.

Use realistic product data, accurate totals, responsive layout, accessible forms, success/error states, and no console errors.
