Build a production-ready project management app called “SprintDock”.

It is a task collaboration platform for product teams. Use React + TypeScript + Tailwind CSS + shadcn/ui. Store all data in memory and do not use external APIs.

Must include:
- Overview dashboard with active projects, overdue tasks, team capacity, burndown-style sparkline, and recent activity.
- Project board with draggable Kanban cards across Backlog, Ready, In Progress, Review, and Done.
- Task detail page with assignee, priority, estimate, checklist, comments, attachments simulation, and activity timeline.
- Roadmap page with milestones, dates, progress, and dependency warnings.
- Team page with workload, availability, roles, and current assignments.
- Settings page with project defaults, statuses, labels, and reset demo data.

Use realistic demo data, validation, responsive UI, success toasts, smooth states, and no console errors.
