Build a production-ready social community app called “CreatorCircle”.

It is a private creator community with content, messaging, and moderation. Use React + TypeScript + Tailwind CSS + shadcn/ui. Store all data in memory and do not use external APIs.

Must include:
- Community home feed with posts, reactions, comments, bookmarks, trending topics, and suggested members.
- Member profiles with bio, content stats, badges, recent posts, and message button.
- Groups page with categories, group cards, join/leave actions, and group detail view.
- Messages page with inbox threads, compose form, unread states, and simulated send.
- Moderation queue with reported posts, reason tags, approve/remove actions, and audit trail.
- Settings page with profile, notifications, privacy controls, and reset demo data.

Use realistic demo data, responsive layout, validation, loading states, smooth interactions, and no console errors.
