Build a production-ready AI support chatbot app called "HelpPilot AI".

It is a customer support assistant workspace with simulated AI responses, knowledge base search, and escalation handling. Use React + TypeScript + Tailwind CSS + shadcn/ui. Store all data in memory, simulate AI locally, and do not call real AI providers or external APIs.

Must include:
- Agent dashboard with open conversations, deflection rate, response quality, escalations, and knowledge gaps.
- Chat workspace with conversation list, customer profile, simulated streaming assistant replies, human handoff, and internal notes.
- Knowledge base page with searchable articles, categories, confidence score, edit simulation, and missing-answer queue.
- Training/evaluation page with test questions, expected answer, simulated score, and improvement suggestions.
- Escalations page with priority, owner, SLA, status transitions, and resolution notes.
- Settings page with provider placeholders, tone controls, routing rules, and reset demo data.

Use clear local-simulation language, realistic support data, responsive layout, accessible chat controls, validation, success toasts, and no console errors.
