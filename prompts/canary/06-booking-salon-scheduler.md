Build a production-ready booking app called “LuxeBook Studio”.

It is an appointment scheduling platform for a premium salon. Use React + TypeScript + Tailwind CSS + shadcn/ui. Store all data in memory and do not use external APIs.

Must include:
- Booking dashboard with today’s appointments, revenue, utilization, cancellations, and staff availability.
- Public booking flow with service selection, stylist selection, calendar, time slots, customer info, and confirmation.
- Appointments page with calendar/list toggle, filters, status updates, and reschedule simulation.
- Staff page with stylists, services, schedules, availability, and current bookings.
- Customer profile detail with history, notes, preferences, and upcoming appointment.
- Settings page with business hours, default deposit, service catalog, and reset demo data.

Use validation, success toasts, responsive layout, smooth loading states, and no console errors.
