Build a production-ready marketplace web app called “TradeMatch Pro”.

It is a two-sided contractor marketplace where property managers find vetted vendors. Use React + TypeScript + Tailwind CSS + shadcn/ui. Store all data in memory and do not use external APIs.

Must include:
- Marketplace home with featured vendors, category filters, search, and trust metrics.
- Vendor directory with filters for trade, location, rating, availability, and insurance status.
- Vendor profile detail with services, reviews, pricing examples, availability, and request quote button.
- Buyer request form with validation, urgency, budget, job details, and success toast.
- Vendor dashboard with leads, quote requests, profile completeness, and response metrics.
- Settings page with account placeholders and reset demo data.

Use realistic demo data, responsive cards/tables, polished interactions, loading states, and no console errors.
