Build a production-ready task and habit tracker called "Momentum Loop".

It helps busy professionals manage daily tasks, recurring habits, focus sessions, and weekly progress. Use React + TypeScript + Tailwind CSS + shadcn/ui. Store all data in memory and do not use external APIs.

Must include:
- Today dashboard with tasks, habits, streaks, focus time, and weekly completion progress.
- Task list with add/edit/delete, due date, priority, tags, completion, filters, and empty states.
- Habit tracker with recurring schedule, streak count, skip reason, completion calendar, and detail drawer.
- Focus session timer simulation with start/pause/reset, session notes, and completed-session history.
- Insights page with charts, category breakdown, overdue tasks, and habit consistency.
- Settings page with default priorities, notification placeholders, theme preference, and reset demo data.

Use smooth interactions, validation, optimistic state updates, responsive layout, success toasts, and no console errors.
