Build a production-ready inventory management app called “StockPilot”.

It is a warehouse inventory control system for a specialty retailer. Use React + TypeScript + Tailwind CSS + shadcn/ui. Store all data in memory and do not use external APIs.

Must include:
- Dashboard with stock value, low-stock count, purchase orders, fulfillment rate, and inventory movement chart.
- Inventory table with SKU search, category filters, location, stock level, reorder point, and status badges.
- Product detail page with stock ledger, supplier info, reorder calculator, and adjustment history.
- Purchase orders page with create PO form, line items, vendor selector, totals, and status transitions.
- Receiving workflow with scan simulation, quantity validation, and success toast.
- Settings page with warehouses, suppliers, reorder rules, and reset demo data.

Use realistic numbers, responsive UI, validation, loading/empty states, and no console errors.
