Build a production-ready personal finance tracker called "Budget Harbor".

It helps households track spending, budgets, goals, and cashflow. Use React + TypeScript + Tailwind CSS + shadcn/ui. Store all data in memory and do not connect to banks, payment processors, or external APIs.

Must include:
- Dashboard with net cashflow, spending by category, budget health, upcoming bills, and savings goal progress.
- Transactions page with search, filters, category editing, split transaction simulation, and add transaction form.
- Budgets page with category limits, progress bars, rollover toggle, warning states, and monthly comparison.
- Goals page with savings targets, contribution planner, timeline, and milestone celebrations.
- Bills page with due dates, recurring status, reminders placeholder, and paid/unpaid toggle.
- Settings page with household members, categories, currency display, and reset demo data.

Use accurate financial math, realistic seeded data, clear disclaimers that data is local demo data, responsive charts/tables, validation, and no console errors.
